# Datastore Keyboard Shortcuts

Qualytics provides keyboard shortcuts to speed up common datastore actions. Shortcuts are context-aware — they activate based on the currently selected item in the tree view or detail page.

!!! tip "All Keyboard Shortcuts"
    This page covers datastore-specific shortcuts only. For a complete list of all keyboard shortcuts across the platform, see the [Keyboard Shortcuts](../../keyboard-shortcuts.md){:target="_blank"} page.

!!! note
    Shortcuts are disabled when an input field, dialog, or menu is focused. Sequential shortcuts (e.g., ++r++ then ++c++) have a 2.5-second timeout between keys.

## Operations

Run operations on the currently selected datastore.

| Shortcut | Action |
| :--- | :--- |
| ++r++ then ++c++ | Run **Sync** operation |
| ++r++ then ++p++ | Run **Profile** operation |
| ++r++ then ++s++ | Run **Scan** operation |

## Datastore Context

Available when a datastore is selected.

| Shortcut | Action |
| :--- | :--- |
| ++e++ | Edit datastore |
| ++a++ then ++t++ | Assign tags |
| <span class="os-mac">++option+f++</span><span class="os-other">++alt+f++</span> | Toggle favorite |
| <span class="os-mac">++cmd+backspace++</span><span class="os-other">++ctrl+delete++</span> | Delete datastore |
