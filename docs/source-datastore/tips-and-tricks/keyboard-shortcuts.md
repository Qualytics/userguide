# Datastore Keyboard Shortcuts

Qualytics provides keyboard shortcuts to speed up common datastore actions. Shortcuts are context-aware — they activate when a datastore is selected in the **tree view** (left sidebar navigation) or when you are on a **datastore detail page** (the overview page of a specific datastore).

!!! tip "Command Bar"
    Press <span class="os-mac">++cmd+k++</span><span class="os-other">++ctrl+k++</span> to open the **Command Bar** — a quick-access search palette where you can find and execute any datastore action by name.

!!! tip "All Keyboard Shortcuts"
    This page covers datastore-specific shortcuts only. For a complete list of all keyboard shortcuts across the platform, see the [Keyboard Shortcuts](../../keyboard-shortcuts.md){:target="_blank"} page.

!!! note
    Shortcuts are disabled when an input field, dialog, or menu is focused. Sequential shortcuts (e.g., ++r++ then ++c++) must be pressed in quick succession.

## Operations

Available when a datastore is selected in the tree view or when viewing a datastore detail page.

| Shortcut | Action |
| :--- | :--- |
| ++r++ then ++c++ | Run **Sync** operation |
| ++r++ then ++p++ | Run **Profile** operation |
| ++r++ then ++s++ | Run **Scan** operation |

## Datastore Context

Available when a datastore is selected in the tree view or when viewing a datastore detail page.

| Shortcut | Action |
| :--- | :--- |
| ++e++ | Edit datastore |
| ++a++ then ++t++ | Assign tags |
| <span class="os-mac">++option+f++</span><span class="os-other">++alt+f++</span> | Toggle favorite |
| <span class="os-mac">++cmd+delete++</span><span class="os-other">++ctrl+delete++</span> | Delete datastore |
