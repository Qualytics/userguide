# Observability Filters

Filters help narrow down the displayed observability checks, making it easier to organize and focus on relevant results.

![observability](../assets/observability/filter.png)

| No. | Filter | Description |
|----|--------|-------------|
| 1. | **Tags** | Filters checks based on tags assigned to tables or files (for example, finance, sales, or production). |
| 2. | **Rule Type** | Filters checks based on the rule category, such as **Volumetric** and **Freshness**. |
